# TIEM-EventSphere
Gateway to College Events

 TIEM EventSphere is a robust, responsive web portal developed to centralize and automate event management for the Techno Institute of Engineering & Management.
 Built using HTML5, CSS3, and JavaScript, the platform features dynamic event listing, secure registration forms, and an administrative dashboard with persistent browser storage.
 The system streamlines the student experience while providing faculty with real-time data tracking, significantly enhancing institutional engagement and extracurricular organization.
